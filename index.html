生産物事故における発生時の対応フロー

■ 基本行動原則

1. 安全の確保を最優先する: 自身の安全、お客様の安全を第一に確保する
2. 作業を即時中断する: 事態の悪化を防ぐため、直ちに作業を止める
3. 誠意をもって正直に報告する: 隠さず、言い訳せず、状況を正確に報告する
4. 単独で判断・約束しない: 補償や今後の対応について、独断や約束は絶対にしない

交通事故・車両トラブル発生時の対応フロー

■ 基本行動原則

1. 人命救助を最優先する: 負傷者がいる場合は、何よりもまず救護活動を行う
2. 二次災害を防止する: ハザードランプや発煙筒などで後続車に危険を知らせ、安全な場所に避難する
3. 警察に必ず通報する: 事故の大小にかかわらず、必ず警察（110番）に連絡し、指示を仰ぐ
4. その場で示談しない: 補償や過失割合について、その場での独断や約束（示談）は絶対にしない

車両トラブル（事故以外）の場合:

  • 警察への通報は不要です
  • 安全確保を最優先してください
  • 東京海上日動へ連絡(0120-119-110)
  • 会社への報告を忘れずに行ってください

災害発生時の対応フロー

■ 基本行動原則

1. 人命救助を最優先する: 自身の安全確保はもちろん、周囲に助けを必要とする人がいれば、可能な範囲で協力する
2. 二次災害を防止する: 周囲の状況をよく確認し、さらなる危険を回避する行動をとる
3. 正確な情報を入手する: ラジオやスマートフォンなどを活用し、公的機関からの信頼できる情報に基づいて行動する
4. むやみに移動しない: 安全が確認できない状況での移動は避け、安全な場所で待機する

熱中症発生時の対応フロー

■ 基本行動原則

「予防は治療に勝る」- 少しでも異変を感じたら、早めに対処することが重症化を防ぐ鍵となります。

1. 体調の変化を最優先する: 自身の体の変化に注意を払い、無理をしない
2. 二次災害を防止する: 症状が出た際は、速やかに作業を中断し、安全な場所へ移動する
3. 正確な情報を入手する: 自身の症状や周囲の状況を冷静に把握する
4. 一人で無理をしない: 症状が改善しない場合や判断に迷う場合は、助けを求める
